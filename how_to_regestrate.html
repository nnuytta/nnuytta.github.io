Как зарегистрировать программу

Для регистрации программы необходимо прислать продавцу серийный номер винчестера. Его можно посмотреть, используя меню VetCAD++ -> О программе. В открывшемся диалоговом окне есть поле "Код машины" - это и есть номер винчестера. Код машины представляет собой положительное девятизначное число. Этот код не меняется при переустановке операционной системы и даже при форматировании вашего жесткого диска.

Если программа выдает отрицательный номер, значит ей не удалось по каким-то причинам получить номер жесткого диска. Отрицательный номер означает, что программа может быть привязана к сетевой карте. Можно попытаться все же получить номер винчестера. Как это сделать, читайте здесь.

На основе этого номера будет сформирован файл лицензии vetcad++.lic, который должен прислать вам продавец программы. Этот файл лицензии необходимо скопировать в папку VetCAD++. После этого утилиты VetCAD++ будут работать в полном режиме. Если возникли проблемы с регистрацией, читайте здесь.
