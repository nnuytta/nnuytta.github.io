Сетевая установка VetCAD++

1. Папку VetCAD++ необходимо поместить на общедоступном сетевом ресурсе, открыть пользователям к ней доступ на чтение (некоторым можно дать доступ на запись для обновления баз например)

2. Установить VetCAD++ как описано здесь, указывая файл SetupVetCAD*.* из сетевой папки.

В случае сетевой установки удобно производить обновления программы, также базы к программе являются общими для всех. В лицензионный файл включаются все номера машин и файл хранится в папке VetCAD++ на сервере. При загрузке VetCAD++ каждый экземпляр программы находит в файле лицензии свой номер.

В случае сетевой установки настройки пользователя (файл vetcad++.ini) будет располагаться в папке Windows на локальной машине пользователя.

Менеджер лицензий VetCAD++.

Возможна установка на сервер менеджера лицензий VetCAD++, для которого создается лицензионный файл, в котором прописано количество одновременно работающих пользователей. В этом случае нет необходимости снимать номера с машин пользователей, достаточно номер сервера. Такой вариант лицензирования возможен при покупке 10 и более рабочих мест. Для этого:

1. Необходимо закачать менеджер лицензий VetCAD++ по ссылке.

2. Запустить файл VetCADLicMan.exe на том сервере, который будет раздавать лицензии (это не обязательно та же машина, на которой находится папка VetCAD++, это может быть любая другая машина в сети). Менеджер сообщит, что лицензия не найдена. Вышлите продавцу "Код машины", который выдал менеджер. Закрыть менеджер можно, нажав сначала на кнопку "Старт!", затем появится кнопка "Выход".

3. Полученный от продавца файл лицензии поместите в папку, в которой находится VetCADLicMan.exe.

4. Подправьте файл LicManager.ini, а именно IP адрес машины, которая будет раздавать лицензии (на которой был запущен VetCADLicMan.exe). Можно изменить порт со стандартно прописанного "1234" на любой другой, если этот занят.

5. Скопируйте LicManager.ini в папку VetCAD++\App на сервере.

6. Запустите VetCADLicMan.exe и нажмите кнопку "Старт!". Менеджер работает и будет раздавать лицензии на количество рабочих мест, прописанное в лицензии.

Лицензии являются плавающими и освобождаются при выходе из AutoCAD (или другой САПР).

Менеджер лицензий как служба

По вышеописанной схеме менеджер работает как приложение и в этом случае нельзя делать выход из системы (Logout), иначе приложение будет выгружено из памяти. Если Вас такой вариант не устраивает, можно воспользоваться службой. Для этого:

1. Выполните 5 пунктов, описанных выше.

2. Зарегистрируйте службу, запустив regservice.bat (см. в составе архива). Проверьте в свойствах службы "VetCAD++ Service", что она запускается автоматически.

3. Подправьте значение параметра Service в файле LicManager.ini на 1 (по умолчанию 0). Получится Service=1.

4. Запустите VetCADLicMan.exe и нажмите кнопку "Активизировать сервис". Теперь менеджер лицензий можно закрыть, раздавать лицензии будет служба. Менеджер лицензий можно только периодически запускать, чтобы посмотреть сколько лицензий захвачено на данный момент.

Примечание: Сделайте запуск службы в автоматическом режиме при старте системы. Менеджер также можно активизировать при старте системы. Для этого создайте bat файл и пропишите в нем путь к VetCADLicMan.exe и через пробел укажите параметр start и поместите bat файл в автозагрузку. Получится что-то вроде D:\vcLicMan\VetCADLicMan.exe start
